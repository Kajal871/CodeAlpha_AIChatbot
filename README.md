# AI Chatbot 🤖

A Java console based chatbot that can have
interactive conversations with users.

## What This Project Does
This chatbot called Buddu understands what you
type and replies in a friendly and smart way.
It recognizes common questions and gives
helpful responses just like a real assistant.

## Features
- Interactive console based chat
- Understands greetings and farewells
- Answers questions about itself
- Tells jokes to make you smile
- Responds to emotions like happiness and sadness
- Handles food, weather and time questions
- Gives a polite reply when it does not understand

## Technologies Used
- Java
- Object Oriented Programming
- Rule Based NLP Logic
- Console Based Interface

## How to Run
1. Clone this repository
2. Open in VS Code
3. Go to src folder in terminal
4. Compile:
   javac ResponseEngine.java ChatBot.java Main.java
5. Run:
   java Main
